#Clarus:
Clarus, meaning clarity â€” is a private web application designed for structured reflection and decision tracking. It helps users log important decisions, track outcomes, and identify thinking patterns over time. Clarus emphasizes privacy, personal growth and clarity by removing distractions like social media feeds, comments or notifications.

App Name: Clarus
Type: Structured Reflection & Decision Tracking Web App
Purpose: Help users log decisions, reflect on outcomes and gain personal clarity
Core Philosophy: No social media, No public validation and Private clarity tool

#Core Features

1. User Profiles: Track personal information and decision activity
2. Decision Logging: Record situations, options, chosen decisions, reasoning and expected outcomes
3. Reflection Module: Review outcomes, lessons learned and insights
4. Insights Dashboard: Visualize decision trends, categories and reflection status

#clarus-backend:

Clarus Backend is the server-side of the Clarus app, built to securely manage users, decisions, reflections and analytics. It provides RESTful APIs for the frontend and ensures a structured, modular and maintainable architecture.

Tech Stack: Node.js, Express.js, MongoDB (with Mongoose), JWT for authentication, bcrypt for password hashing.

Features:

1. User authentication (signup, login, logout)
2. Decision and reflection CRUD operations
3. Dashboard analytics
4. Secure and modular structure (MVC pattern)

Setup:
1.Clone the repo
2.Install dependencies: npm install
3.Set environment variables: MONGO_URI, JWT_SECRET, FRONTEND_URL
4.Start server: npm run dev
